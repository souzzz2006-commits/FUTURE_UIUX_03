# Task 3 – Collaborative Design Platform for Freelancers

This project is part of my UI/UX Internship at Future Interns.

## Task Overview
The objective of Task 3 is to design a collaborative freelancer platform similar to Fiverr or Upwork, specifically tailored for UI/UX designers. The platform allows clients to post jobs, browse designers, view portfolios, and enables designers to submit proposals seamlessly.

## Screens Designed

### Desktop Screens
- Home Page
- Job Listings Page
- Designer Profile Page
- Proposal Submission Page

### Mobile Screens
- Home Screen
- Jobs Screen
- Profile Screen
- Proposal Screen
- Proposal Success Screen

## Key Features
- Job browsing and filtering
- Designer profiles with skills and portfolio showcase
- Proposal submission workflow
- Responsive UI for desktop and mobile
- Interactive Figma prototype
